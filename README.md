# Person Data Management

A robust web application designed to handle, process, and manage personal data efficiently. The project is structured with a dedicated backend API to ensure clean separation of concerns and scalable development.

## 🚀 Technologies

This project leverages the following modern technologies:
- **TypeScript** - For type safety and enhanced developer experience.
- **Node.js** - As the asynchronous event-driven JavaScript runtime.
- **Git & GitHub** - For version control and repository management.

## 🛠️ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Make sure you have installed on your machine:
- [Node.js](https://nodejs.org) (v18 or higher recommended)
- [npm](https://npmjs.com) (comes bundled with Node.js)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   cd person-data
   ```

2. **Navigate to the API folder and install dependencies:**
   ```bash
   cd api-person-data
   npm install
   ```

---

## ⚡ Development

To run the application locally in development mode with live-reloading:

```bash
# Inside the api-person-data directory
npm run dev
```
